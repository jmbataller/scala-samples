## Commands

### SBT

SBT version:
> sbt sbt-version

Compile:
> sbt compile

Package:
> sbt package

Test:
> sbt test

Show project/modules versions:
> show version

Show dependencies classpath
> show dependencyClasspath

Go to a project module
> project <name>
